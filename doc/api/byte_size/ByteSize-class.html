ByteSize

ByteSize class Null safety

ByteSize handles how byte sizes are represented.

var size = new ByteSize(1024);
Implemented types
  • Comparable

Constructors

ByteSize([num byteSize = 0])
Initialize class with the size in Bytes denoted by byteSize.
ByteSize.fromJson(Map<String, Object> entityMap)
Initialize ByteSize object from JSON object according to order of precedence i.e from 'b' -> 'PiB', where 'b' has higher precedence to 'KiB' and 'KiB' has higher precedence to 'KB' and so on.
